# Personal Portfolio Website

Welcome to the repository for my personal portfolio! This is a fully responsive, custom-built static website designed to showcase my journey, projects, and technical skills as an Integrated M.Sc. Computer Science student.

## 🚀 Live Demo
https://rittusoney.github.io/

## ✨ Key Features
* **Dynamic Certificate Fetching:** Integrated the **Google Drive API** using vanilla JavaScript (`fetch()`) to dynamically pull and display my latest certificates directly from a cloud folder.
* **Asynchronous Contact Form:** Implemented an AJAX-powered contact form using **Formspree** and `FormData` to handle user submissions securely without page reloads.
* **Fully Responsive Design:** Utilized CSS Flexbox, Grid, and targeted media queries to ensure a seamless experience across desktop, tablet, and mobile devices.
* **Optimized Caching:** Built client-side caching using `localStorage` to store API responses, drastically reducing load times and API quota usage on subsequent visits.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Styling:** Custom CSS & Tailwind CSS (via CDN)
* **APIs & Integrations:** Google Cloud Console (Drive API v3), Formspree
* **Deployment:** GitHub Pages

## ⚙️ Local Installation
If you'd like to run this code locally:
1. Clone the repository: `git clone https://github.com/yourusername/your-repo-name.git`
2. Open `index.html` in your browser, or use an extension like VS Code Live Server.
3. *Note: You will need to supply your own Google Drive API key in `certs.js` to enable the certificate fetching feature locally.*
